# CommerceProject
## Java 과제 커머스 제작

### 목차
- 프로젝트 소개
- 주요 기능
- 프로젝트 구조
- 빌드/배포 방법


### 프로젝트 소개

- 해당 프로젝트는 Java 입문자로서 커머스 플랫폼을 제작해보며 Java에 적응하도록 돕는 과제로서 제작된 커머스 플랫폼 구현 프로젝트 이다.


### 주요 기능

- 카테고리 목록을 조회하여 선택하는 기능
- 특정 카테고리를 선택했을 때 해당 카테고리의 상품목록을 보여주는 기능
- 특정 상품을 선택했을 때 해당 상품의 정보 및 재고를 보여주는 기능
- 해당 플랫폼을 종료하는 기능
- 상품 목록에서 뒤로 가기를 선택하여 카테고리 목록을 다시 조회할 수 있는 기능


### 프로젝트 구조

- Main :
    - main 함수를 담당하며 조회 확인, 취소를 입력 받아 확인 시 CommerceSystem을 구동하여 플랫폼을 가동하거나, 취소 시 프로그램을 종료한다. 
- CommerceSystem :
    - 사용자의 입력에 따라 Category에서 각 카테고리 별 세부항목을 전달 받고, 전달 받은 내용을 출력해주며, Category의 리스트를 관리하는 등 프로그램의 핵심 역할을 수행한다.
- Category :
    - CommerceSystem으로부터 입력 내용을 전달 받아 각 Product의 정보를 전달하고 Product의 리스트를 관리한다.
- Product :
    - 각 상품의 필드값을 저장하며 프로그램의 뿌리 역할을 담당한다.
- Customer :
    - 고객들의 정보를 관리하며 누적금액을 정산하고 등급의 등락을 결정한다.
